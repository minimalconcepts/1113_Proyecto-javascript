# Guia de creadores de marcas

El apartado de creadores de marcas sirve para explicar quienes fundaron o impulsaron marcas importantes.

## Donde se abre

```text
#/fundadores
```

Tambien aparece en el menu como:

```text
Creadores de marcas
```

## Donde se agregan fundadores

Abre:

```text
assets/js/data.js
```

Busca:

```js
founders: []
```

Agrega una ficha asi:

```js
{
  name: "Nombre del creador",
  brand: "Marca",
  area: "Autos",
  year: "Anio de fundacion",
  description: "Descripcion corta de su importancia."
}
```

## Areas recomendadas

```text
Autos
Motos
Aviones
Motos y autos
```

## Archivos relacionados

Vista:
